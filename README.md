# 2024-DSP-Simulation-NPU
2024西工大数字信号处理实验，仅供参考，切勿抄袭





#### 1. **实验一：时域与频域采样**
- **时域采样**：通过设置不同采样频率（1000Hz、300Hz、200Hz）对信号采样，观察频谱混叠现象，验证**奈奎斯特采样定理**。
- **频域采样**：验证频域采样对时域信号周期延拓的影响，指出采样点数不足会导致时域混叠失真。

#### 2. **实验二：FFT频谱分析**
- 对非周期序列（如矩形、三角波）和周期序列（如余弦信号）进行FFT分析。
- 强调对周期信号进行FFT时，**采样点数应为周期的整数倍**，以避免频谱泄漏。
- 包含对复合周期信号的频谱分析与多分量提取。

#### 3. **实验三：IIR滤波器设计**
- 使用**双线性变换法**设计IIR数字滤波器（低通、高通、带通）。
- 对比**巴特沃斯**与**切比雪夫I型**滤波器性能，发现切比雪夫I型在相同指标下阶数更低、计算更高效。
- 将设计的滤波器应用于多路调幅信号分离，验证滤波效果。

#### 4. **实验四：FIR滤波器设计**
- 使用**窗函数法**（布莱克曼窗）设计FIR低通滤波器。
- 通过**重叠相加法**实现长序列滤波，成功从含噪信号中提取出目标调幅信号。
- 分析了理想滤波器截止频率的选取原则。
